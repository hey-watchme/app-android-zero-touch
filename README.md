# ZeroTouch Android

`ZeroTouch` は、Android タブレットを現場に置くだけで、会話をメモ、ナレッジ、
将来的にはエージェント行動へ変えていくためのアプリです。

現在の実装は「録音 -> 停止 -> 再生」の最小ボイスメモ版です。
今後は `音声取得 -> STT -> ファクト抽出 -> アノテーション -> カード化 -> ナレッジ化`
のパイプラインへ拡張します。

バックエンドは `WatchMe` プロジェクトのインフラを使う前提です。
モデルやデータベースの詳細は後続で定義します。

企画の基準は `docs/ambient-agent-spec.md` を参照してください。
既存の実装メモは `HANDOFF.md` にあります。
